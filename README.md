## 🚀 Jak spustit

**Možnost A — přímo v prohlížeči (bez instalace)**
1. Stáhni soubor `dp-ai-chat.html`
2. Otevři ho dvojklikem v prohlížeči
3. Vlož svůj Anthropic API klíč

**Možnost B — přes GitHub Pages (sdílení s ostatními)**
1. Forkni nebo nahraj tento repozitář na GitHub
2. Jdi do **Settings → Pages → Source: Deploy from branch → main → / (root)**
3. Za pár minut běží na `https://TVOJE-JMENO.github.io/NAZEV-REPO/dp-ai-chat.html`

---

## 🔑 API klíč

Aplikace potřebuje vlastní Anthropic API klíč. Klíč se ukládá **pouze do localStorage prohlížeče** — nikam se neodesílá, není v kódu.

1. Zaregistruj se na [console.anthropic.com](https://console.anthropic.com)
2. Klikni na **API Keys → Create Key**
3. Zkopíruj klíč (začíná `sk-ant-`) a vlož ho do aplikace

> ⚠️ **Nikdy nedávej svůj API klíč do kódu!** Každý uživatel zadává svůj vlastní klíč.
